El programa muestra una lista doblemente enlazada implementada en Python con un menú interactivo que da las opciones de:
   Agregar al final
   Agregar al inicio
   Mostrar lista
   Eliminar por carnet
   Graficar lista
   Salir
   
Hay un archivo PNG llamado lista_doble_enlazada en donde esta el ejemplo de una inserción de datos que realicé yo y luego la grafiqué con la libreria de Graphviz y utilizando la funcion del codigo llamada graficar, en donde me guié de un video para poder implementarla dentro de mi codigo.
